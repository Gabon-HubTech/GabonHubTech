# Gabon HubTech Client

Plateforme web communautaire pour les passionnés de technologie au Gabon.

## Technologies
- Next.js 16 (App Router)
- React 19
- Tailwind CSS
- Framer Motion

## Installation

1. Cloner le repo : `git clone <url>`
2. Installer les dépendances : `npm install`
3. Configurer les variables d'environnement (`.env.local`) :
   - `NEXT_PUBLIC_API_URL`
   - `NEXT_PUBLIC_ADMIN_PASSWORD`
4. Lancer le serveur de développement : `npm run dev`
5. Pour le build de production : `npm run build`

## Fonctionnalités principales
- Formulaire de contact sécurisé
- Gestion dynamique des événements et formations
- Interface d'administration pour la gestion de contenu
